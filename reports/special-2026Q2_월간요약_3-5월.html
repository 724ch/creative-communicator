5 META PATTERNS

분기의 다섯 메타 패턴

세 달을 가로지르는 발견된 패턴 다섯. 패턴 5(카피라이팅이 디자인을 견인)는 분기 전체에서 다섯 차례 확인되며 *가장 강함*.

PATTERN 01

한 구조의 추상도 누적

3/31 그·ㄹ (음운) → 4/03 아름+다움 (미학) → 5/21 알음+다움 (인식론). 같은 *3원 구조*가 음운 → 미학 → 인식론으로 추상도 한 단계씩 상승.

PATTERN 02

디학 환경이 학계로 확장

3/22 디학 수업 → 5/19 디자인사학회 특강(김의래). 환경이 *교실*에서 *학회*로 한 단계 넓어짐. 디학이 그 다리.

PATTERN 03

외부 자료 → 별도 리포트 → 흡수 보류

4/23 퍼스 귀추법 (4월 일지) → 5/19 김의래 special- → 5/21 김정운 special-. 흡수 결정의 *시간 간격*이 정확도를 보장.

PATTERN 04

명명 → 룰 → 도구의 3단 순서

4월 명명(정체성·언어화실) → 5월 룰(세 척추) → 5월 도구(스킬 4·토이 5). 명명 없이 룰이 안 박히고, 룰 없이 도구가 산발.

PATTERN 05 · STRONGEST

카피라이팅이 디자인을 견인한다

3/22 SMR "구구절절함을 믿어보자" → 4/23 한 줄 카피의 미시 결정 → 5/15 멋짓 → 5/16 Melt 컨셉 단어 → 5/21 알음+다움. 한 줄·한 단어가 시스템 전체를 정렬한다는 패턴이 분기 전체에서 다섯 차례 확인. 분기의 *가장 강한* 횡단 패턴.
